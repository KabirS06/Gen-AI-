# OPTUNA 
Optuna is an automatic hyperparameter optimization software framework, particularly designed for machine learning. It features an imperative, define-by-run style user API. The code written with Optuna enjoys high modularity, and the user of Optuna can dynamically construct the search spaces for the hyperparameters.
It also allow user hypertune different model like SVM , RandomForest ,GradientBoosting and others parallaly with their respective parameters
It also allows Distibuted Computing and is easily integrated  with frameworks like sckit-learn , tensorflow , pyTorch , mlflow etc.  

Optuna used some key terminologies within the frameworks . Some of them are:
1. Study 
2. Trial 
3. Trial Parameters
4. Objective Function
5. Sampler (TPESampler() , RandomSampler() , GridSampler() )
